Command line installation

Once installed, OONI Probe runs tests automatically every day. For details, read the OONI Probe CLI user guide.

{{/* Debian / Ubuntu */}}

Debian / Ubuntu

  1. 1. Add the GPG signing key. 
    sudo gpg --no-default-keyring --keyserver hkp://keyserver.ubuntu.com \
    --keyring /etc/apt/keyrings/ooni-apt-keyring.gpg \
    --recv-keys 'B5A08F01796E7F521861B449372D1FF271F2DD50'
  2. 2. Add the OONI apt repository.

    Choose HTTP (recommended) or Tor.

    echo "deb [signed-by=/etc/apt/keyrings/ooni-apt-keyring.gpg] https://deb.ooni.org/ unstable main" \
    | sudo tee /etc/apt/sources.list.d/ooniprobe.list
    Use Tor instead 
    sudo apt-get install tor apt-transport-tor
  echo "deb [signed-by=/etc/apt/keyrings/ooni-apt-keyring.gpg] tor+http://deb.ooni.org/ unstable main" \
    | sudo tee /etc/apt/sources.list.d/ooniprobe.list
  3. 3. Install OONI Probe. 
    sudo apt-get update
  sudo apt-get install ooniprobe-cli
{{/* macOS */}}

macOS (Homebrew)

  1. 1. Install Homebrew.

    Follow the instructions at brew.sh.

  2. 2. Install OONI Probe. 
    brew install ooniprobe
  3. 3. Enable daily automatic runs. 
    ooniprobe autorun start
